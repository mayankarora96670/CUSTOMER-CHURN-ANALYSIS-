# Customer Churn Analysis

A simple data analytics project that explores customer churn using Python and Power BI. The goal is to identify churn patterns, understand customer behavior, and generate insights that can help improve customer retention.

## Project Overview

This project includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Churn pattern analysis
- Interactive Power BI dashboard

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

## Key Insights

- Customers with month-to-month contracts have the highest churn rate.
- Longer customer tenure is associated with lower churn.
- Customers using automatic payment methods are less likely to churn.
- Contract type and service usage significantly influence churn behavior.

## Dashboard Highlights

- Churn vs. Non-Churn distribution
- Churn by contract type
- Churn by payment method
- Churn by tenure
- Customer demographics and service usage analysis
